# ``RecordCatalogKit``

A concurrency-safe Swift SDK for the Discogs API.

## Overview

Create a ``RecordCatalogClient`` with a unique User-Agent and the authentication
mode appropriate for the application. Use its database, user, marketplace,
inventory, and image services from asynchronous Swift tasks.

For complete examples, authentication guidance, endpoint coverage, and API
terms reminders, see the package README.
